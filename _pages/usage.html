Usage

Using LESS

less 
.foo{
    .pttrn-hexagon();
}

.bar{
    .pttrn-hexagon(#ED5821);
}
html 
<div class="foo">
    pttrn
</div>

<div class="bar">
    pttrn with color
</div>

Using CSS

html 
<div class="pttrn-hexagon">
    pttrn
</div>

Color

Default color is #333333, but you can change .pttrn color from here.

pttrn.min.css
